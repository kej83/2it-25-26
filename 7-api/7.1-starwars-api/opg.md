# Nettside med planetinfo
Lag nettside med to knapper. En knapp viser alle planeter med litt info om hver planet, f.eks. navn, terreng og populasjon (name, terrain, population).

Den andre knappen lar deg søke på en planet (input-felt). Hvis planeten blir funnet, skal du vise så mye informasjon om planeten som du klarer.

Bruk nettsiden:
[swapi.info/planets](https://swapi.info/planets)
